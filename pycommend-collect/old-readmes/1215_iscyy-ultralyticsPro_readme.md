## 使用🍉

### 安装

在**Python>=3.9.0** 的环境中克隆版本仓并安装 requirements.txt，包括**PyTorch>=1.7**。

```bash
$ cd ultralyticsPro
$ pip install -r res.txt  # 安装
```

### 训练

```bash
$ python train.py ultralytics/cfg_yolo11/YOLO11/yolo11.yaml
```

### 推理

`detect.py` 在各种数据源上运行推理, 并将检测结果保存到 `runs/detect` 目录。

```bash
$ python predict.py
```
